# 星舰猎盘产品路线图

星舰猎盘是一个模块化宏观交易研究平台。各研究模块共享数据协议、信号协议、报告渲染和版本追踪能力，但可以独立开发与发布。

## 平台分层

1. 数据舱：官方 API、人工导入、原始快照与数据质量检查。
2. 计算舱：确定性公式、规则信号与可回归测试的指标。
3. 分析舱：结构化 AI 解读，只引用已经验证的数据。
4. 报告舱：网页、长图和 PDF 模板。
5. 任务舱：定时更新、人工审核、归档和告警。

## 模块路线

- M1 `cpi-research`：CPI 分项、趋势信号、跨资产路径。
- M2 `fomc-radar`：利率路径、点阵图、声明和发布会语言变化。
- M3 `jobs-monitor`：非农、失业率、工资与劳动参与率。
- M4 `liquidity-map`：TGA、逆回购、美元流动性与金融条件。
- M5 `event-console`：宏观日历、预期差、历史事件复盘。

## CPI Research 里程碑

- v0.1：演示数据、计算函数、规则信号和研究页面。
- v0.2：BLS API、series ID 配置、原始数据快照和数据校验。
- v0.3：结构化 LLM 分析与引用检查。
- v0.4：历史月份切换、发布前后对比和 PNG/PDF 导出。
- v1.0：自动更新、人工审核、报告归档和异常告警。
